# Bank Management System

A Bank Management System developed using Django and MySQL. The application provides banking operations for customers and administrative controls for admins.

## Features

### Customer Features
- User Registration
- User Login
- Check Account Balance
- Deposit Money
- Withdraw Money
- Submit Loan Requests
- Request ATM Card
- Request Checkbook
- View Admin Responses

### Admin Features
- Admin Login
- View All Users
- Search User by ID
- Delete Users
- View Customer Requests
- Respond to Requests
- Approve Requests
- Reject Requests

## Technologies Used

- Python
- Django
- MySQL
- HTML
- CSS
- Git
- GitHub
